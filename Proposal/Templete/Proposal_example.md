# Game Introduction

《泰拉瑞亞》是冒險之地！是神秘之地！是可讓你塑造、捍衛、享受的大地。在泰拉瑞亞，你有無窮選擇。手指發癢的動作遊戲迷？建築大師？收藏家？探險家？每個人都能找到自己想要的。

從建造基本的藏身之處開始，接著挖掘礦石及其他資源。探索並製作超過500 種各式魔法、遠程、近戰、和召喚武器，以及盔甲，藉助它們與數百種不同的敵怪戰鬥。很快，你將直面超過20 個巨大的Boss。還可以去釣魚、騎上坐騎、尋找漂浮島、為有用的NPC建造房屋、釋放你的創造力並建造美麗的建築，有很多很多事情可以做。
