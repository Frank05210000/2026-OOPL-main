# Game Introduction

這是一款類似於 Minecraft 的 2D 沙盒遊戲。
從建造基本的藏身之處開始，接著挖掘礦石及其他資源。探索並製作超過500 種各式魔法、遠程、近戰、和召喚武器，以及盔甲，藉助它們與數百種不同的敵怪戰鬥。很快，你將直面超過20 個巨大的Boss。還可以去釣魚、騎上坐騎、尋找漂浮島、為有用的NPC建造房屋、釋放你的創造力並建造美麗的建築，有很多很多事情可以做。


遊戲畫面參考：
https://www.youtube.com/watch?v=_Q6NZSJtq8U


遊戲流程：
 1. 初期探索：挖掘金礦或銀礦，合成護甲與武器，並擊敗 克蘇魯之眼。
 2. 腐化之地：使用炸彈或淨化粉末深入地下，摧毀暗影珠並召喚、擊敗 世界吞噬者，收集魔晶礦製作 暗影套裝。
 3. 地牢決戰：前往地圖邊緣的地牢，在夜晚擊敗 骷髏王，進入地牢取得寶箱武器（如 村正大刀 或 藍月）。
 4. 地獄終點：挖掘至地圖最底層，採集 地獄石 與 黑曜石，製作 熔岩套裝，作為遊戲後期的裝備目標。


符合老師所提到的「會死亡」、「會獲勝」、「有關卡」條件
